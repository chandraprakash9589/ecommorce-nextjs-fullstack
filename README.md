## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```


run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Add the .env file
MONGO
AUTH_SECRET
AUTH_URL
GITHUB_ID
GITHUB_SECRET
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
